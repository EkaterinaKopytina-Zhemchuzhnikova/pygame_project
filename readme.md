Проект PyGame под названием «Морской бой», который обладает следующими функциональными возможностями:

- Расстановка кораблей игрока по нажатию на клавиши 1, 2, 3 или 4 (в зависимости от количества палуб корабля), клавиш "вправо"/ "вниз" (в зависимости от расположения корабля) и клавиш мыши для непосредственного располажения корабля на поле.
- Расстановка кораблей на поле в соответствии с картой, которая будет получена из текстового файла.
- Выбор игрока.
- Получение информации о правилах игры.

# Приступая к работе
Основной модуль проекта board.py, находится в корневой папке проекта-именно его требуется запустить.

# Предварительные условия
Библиотеки, которые необходимы для корректного функционирования программы описаны в requirements.txt, который расположен к корневой папке проекта.

# Структура проекта

Основной модуль проекта board состоит из классов:

- MyBoard, наследника  Board - отвечает за игровое поле игрока-человека. Основные методы: attack()-отвечает за выстрел ИИ, next_shot()-выстрел ИИ при наличии раненного корабля, mark_neighbors()-отмечает соседние точки убитого ИИ корабля, on_click()-расстановка кораблей игрока.

- EnemyBoard, наследника Board-отвечает за поле ИИ. Основные методы: on_click()-выстрел человека по рандомно расставленным кораблям ИИ методом take_a_cage().  

- Tile-отвечает за расстановку в соответствии с картами, находящимися в текстовых файлах.

- Baloon - отвечает за генерацию шариков на финальной заставке игры.

  Функции  load_music() и load_image() нужны для загрузки музыки и картинок соответственно. Функция start_screen()-помогает оформить стартовую заставку игры. Функция choose_hero()-помогает осуществить выбор героя для игры. Функция total_play()-выводит счет игры, а функция congradulations()-выводит сообщение об итоге игры.

  В папке data лежат вспомогательные материалы: картинки, музыка и текстовые файлы с картами.
